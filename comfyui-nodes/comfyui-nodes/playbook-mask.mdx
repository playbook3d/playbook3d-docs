## Overview

The **Playbook Mask** node fetches the **Mask Render Pass** from Playbook3D using an API key and an optional `run_id`. It applies optional Gaussian blur and returns both the composite mask and individual color-based masks.

## ‚öôÔ∏è Functionality

- Uses `api_key` to authenticate with Playbook3D.
- If `run_id` is provided, fetches the Mask Pass for that specific render.
- If no valid render is found, returns the **default mask**.
- Extracts **8 color-based masks** from the composite image.
- Applies optional **Gaussian blur** to smooth the masks.
