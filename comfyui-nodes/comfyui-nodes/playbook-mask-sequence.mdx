## Overview

The **Mask Pass Sequence** node fetches a ZIP file containing multiple **Mask Render Pass** images from Playbook3D. It extracts and returns both the composite mask and 8 individual color-based masks as a batch.

## ‚öôÔ∏è Functionality

- Uses `api_key` to authenticate with Playbook3D.
